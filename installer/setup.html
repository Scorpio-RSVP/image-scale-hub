Connect to Your Server

Use the web console provided by your hosting provider, or use an SSH client:

  • Windows: Use PuTTY or Windows Terminal
  • Mac/Linux: Open Terminal and type: ssh root@YOUR_SERVER_IP
3

Run the Installer

Copy and paste this single command:

curl -sSL https://raw.githubusercontent.com/YOUR_USERNAME/image-resizer-pro/main/install.sh | sudo bash

The installer will ask you a few questions:

  • Your app name (e.g., "My Photo Tool")
  • Domain name (optional)
  • Admin username & password
4

Access Your App!

Once complete, visit http://YOUR_SERVER_IP:5000 in your browser.

‚ùì Frequently Asked Questions

Do I need to know how to code?

No! The one-click deploy options and our installer script handle everything automatically. Just answer a few simple questions.

How much does hosting cost?

You can start for free with Railway or Render. For production use, a $4-5/month VPS is recommended.

Can I use my own domain name?

Yes! The installer can set up your domain with free SSL certificates automatically.

Is my data safe?

Yes. All data stays on your server. We don't have access to your images or user data.

How do I get help?

Open an issue on GitHub or join our community discussions. We're happy to help!
